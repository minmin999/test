5- Installation: Follow the below instruction to set up your theme structure:
        Once you activate the theme by default the post will be set as your homepage.
        1.In order to set the homepage create a new page as Home page and assign it "custom-home-page" page template and publish your page.
        2.Now go to Settings->Readings->Front page displays and choose your home page that you have created and save it.
        3.Now your page is setup all you need to do is set the slider and the content as per as screenshot given through                                
        customizer(Appearance->customize->vw settings).
