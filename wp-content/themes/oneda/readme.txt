=== Description ===
Oneda is a clean, multipurpose one page WordPress theme that can help you build an awesome website easily by drag and drop. The theme comes with 17+ pre-built front page sections, including Testimonials, team, pricing, Google Map, portfolio, gallery, call to action and HTML5 video background etc. In each section, you can customize its background image, Google Fonts, color, overlay, font size, button radius and many others, so your website will be unique and awesome. And the theme also has beautiful parallax scrolling and animation effects. Built on Bootstrap framework, Oneda is fully compatible with various devices. Moreover, the theme is compatible with WooCommerce and Contact Form 7. Demo here: https://www.coothemes.com/demos/theta-all/?page_id=147
